# Coin Hub 移除登录 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 彻底移除 Coin Hub 的登录、会话和权限校验，让用户默认直接进入控制台，并用新的测试覆盖免登录链路。

**Architecture:** 先用测试把入口行为和壳层文案改成“免登录”预期，再删除登录相关代码和环境依赖，最后更新文档与全量验证。受保护布局保留文件位置，但职责缩减为纯页面壳层，以减少路由结构扰动。

**Tech Stack:** Next.js App Router、React 19、TypeScript、Vitest、Playwright
