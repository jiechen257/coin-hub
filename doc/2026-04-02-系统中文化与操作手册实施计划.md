# 系统中文化与操作手册 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 Coin Hub 的用户界面统一为中文，并新增一份面向使用者的操作手册文档。

**Architecture:** 直接在现有页面和组件中替换用户可见文案，并在少量需要展示状态枚举或示例数据的位置补充中文映射。保持现有路由、接口和数据结构不变，通过测试更新和手册补充收口。

**Tech Stack:** Next.js 16、React 19、TypeScript、Vitest、Playwright、Markdown
